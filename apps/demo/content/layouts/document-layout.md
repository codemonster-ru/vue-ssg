# Document Layout

`VfDocumentLayout` is the main layout primitive for documentation pages.

## Sidebar + Content + Aside

Use the three-column mode when the page has:

- left navigation
- central article content
- right-side table of contents

## Sticky Regions

Enable `sticky-header`, `sticky-sidebar` and `sticky-aside` when the docs site
needs a persistent navigation frame while the article scrolls.

## Mobile Behavior

Use `hide-sidebar-on-mobile` and `hide-aside-on-mobile` to keep the reading
experience simple on smaller screens.
